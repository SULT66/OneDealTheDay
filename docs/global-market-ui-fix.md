# Global market UI fix

This change replaces the unreadable native market selector with a custom accessible dropdown that displays country flags, names, currencies, selected-state highlighting, and dark-mode-safe colors.

Selecting a market now updates the URL, persistent preference, page language metadata, market banner, hero copy, Top 10 heading, and product-card market context without pretending that USD retailer prices have already been converted.

Product and featured-card actions are aligned to a consistent bottom row on desktop and a full-width vertical layout on mobile.
